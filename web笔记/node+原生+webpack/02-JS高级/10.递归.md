概念：自己调用自己

类似于循环的效果，可以让一段代码重复的执行下去。

语法：

```js
function fn(){
    //.....对应的操作
    fn()
}
fn()
```

案例：输出10-1；

```js
for(let i = 10 ; i > 0 ; i--){
    console.log(i)
}

//递归版本
function fn(n){
    console.log(n); //10
    n--;  //9
    if(n > 0){
        fn(n)
    }
}
fn(10)
```

递归的分析流程：

1. 需要分析代码是否有重复执行的代码，
2. 调用函数自身
3. 有一个判断用于结束递归调用，给一个出口
