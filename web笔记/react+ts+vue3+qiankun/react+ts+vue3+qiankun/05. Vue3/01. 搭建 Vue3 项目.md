# 搭建 Vue3 项目

## 一、Vue 的发展

1. 2014 年，Vue1.0 发布；
2. 2016 年，Vue2.0 发布；
3. 2020 年 09 月，Vue3.0 发布；

## 二、Vite 搭建 Vue3 项目

### 1、创建项目

```bash
yarn create vite chitu-project --template vue-ts
```

### 2、下载依赖包

```bash
cd chitu-project
yarn
```
