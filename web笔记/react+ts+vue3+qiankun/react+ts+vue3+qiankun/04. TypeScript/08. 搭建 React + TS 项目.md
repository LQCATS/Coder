# 搭建 React + TS 项目

## 一、创建项目

### 1、使用 Vite

```bash
yarn create vite chitu-project --template react-ts
```

### 2、使用 CRA

```bash
npx create-react-app chitu-project --template typescript
```

## 二、下载其他插件

```bash
# 路由插件
yarn add react-router-dom
# 状态机插件
yarn add @reduxjs/toolkit react-redux
# antd
yarn add antd
# less
yarn add less -D
# 网络请求
yarn add axios
```
